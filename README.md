Кейс-задача 4: Максимальная сила драконьей стаи




Описание задачи

Известно, что сила дракона определяется количеством его голов.
Если в стае несколько драконов, сила стаи определяется произведением количества голов у каждого дракона.

Например:

3 дракона: 3, 4 и 5 голов
Сила стаи = 3 × 4 × 5 = 60

В задаче известно общее количество голов N, принадлежащих всей стае.

Необходимо определить максимально возможную силу стаи, если:

суммарное количество голов равно N

у одного дракона может быть не более 7 голов

Входные данные

Одно натуральное число:

N

где:

0 < N < 100

— общее количество голов драконьей стаи.

Выходные данные

Программа должна вывести максимально возможную силу стаи, которую можно получить при разбиении N голов на драконов (каждый дракон имеет от 1 до 7 голов).

Алгоритм решения

Для решения задачи используется динамическое программирование.

Создаётся массив dp, где:

dp[i] — максимальная сила стаи при i головах

Алгоритм работы программы:

Вводится число N.

Создаётся массив dp размером N + 1.

Начальное значение:

dp[0] = 1

Для каждого числа i от 1 до N перебираются возможные значения голов одного дракона (от 1 до 7).

Для каждого варианта вычисляется произведение:

dp[i - heads] * heads

Выбирается максимальное значение.

Итоговый результат находится в dp[N].
